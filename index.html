MLflow Docker Project
Bitcoin Hourly Price Predictor

This end-to-end machine learning project predicts whether Bitcoin (BTC/USD) will go UP or DOWN in the next hour. Built with Docker and MLflow, it features a fully containerized workflow: data fetching, preprocessing, training, and experiment tracking — no API keys required.

Key Features:

  • Fetches hourly BTC OHLC data via Yahoo Finance
  • Cleans and transforms data for modeling

  • Uses Docker Compose to coordinate containerized stages

Why It’s Valuable:

This project demonstrates how to operationalize machine learning with containerized components, reproducible runs, and automated tracking. It’s ideal for showcasing real-time inference and MLOps foundations.
