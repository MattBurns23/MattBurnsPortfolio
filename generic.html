Cryptocurrency Data Analysis & Visualization

This project leverages the CoinMarketCap API to fetch real-time cryptocurrency data, process it using Python (Pandas), and generate insightful visualizations with Seaborn and Matplotlib. The data includes price trends, percentage changes, and historical analysis for major cryptocurrencies like Bitcoin, Ethereum, and XRP.

Features:

  • Fetches live crypto data via API calls
  • Cleans and structures data for analysis
  • Visualizes price trends and percentage changes
  • Stores data in CSV format for further analysis
  • Uses Seaborn for professional, interactive charts

Why It’s Valuable:

This project demonstrates API integration, data processing, and data visualization skills, making it a great showcase for data analytics and financial technology applications.

Movie Data
Analysis & Cleaning

This project focuses on analyzing and cleaning a dataset of movies using Python, Pandas, and Seaborn. The dataset includes key movie attributes such as ratings, genres, release years, scores, votes, budget, and gross earnings. The project involves data preprocessing, handling missing values, converting data types, and creating new columns to enhance analysis.

Key Features:

  • Loads and explores a movie dataset using Pandas
  • Identifies and handles missing or inconsistent data
  • Converts numerical columns to appropriate data types
  • Extracts and creates a corrected year column from the release date
  • Prepares the dataset for further statistical analysis and visualization

Why It’s Valuable:

This project demonstrates data cleaning and transformation techniques, which are essential for data analytics and machine learning workflows. By refining raw movie data into a structured format, it enables deeper insights into trends in movie performance, budgeting, and audience reception.

Amazon Web Scraper Project

This project automates the process of tracking product prices on Amazon using Python. By leveraging BeautifulSoup and Requests, the script extracts product details such as title and price, storing them in a CSV file for historical analysis. The program runs on a loop, continuously checking for price updates and appending new data. This project showcases the power of web scraping for e-commerce data monitoring and price trend analysis.

Key Features:

  • The script pulls product data, including the title and price, by parsing HTML content using BeautifulSoup.
  • Extracted data is saved to a CSV file, allowing for easy tracking and analysis of price trends over time.
  • The script is designed to run continuously and append new price data at regular intervals.
  • The data is visualized and managed with Pandas, providing a clear view of historical pricing data for further analysis.

Why its Valuable:

This project is an excellent demonstration of web scraping, data automation, data extraction, CSV storage.
